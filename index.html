截图识别（DeepSeek）

填入你自己的 DeepSeek API Key 后，点击列表上方「📷」即可从订单 / 小票截图识别物品名称、价格、日期，结果会预填到表单供你确认。 识别分两步：① 在你的浏览器本机用 OCR 提取文字；② 仅把提取出的文字发送给 DeepSeek 整理成结构化信息——图片不会离开本机。
⚠️ 需要联网；首次使用会从 CDN 下载 OCR 组件与中文识别数据（约数 MB）。Key 仅保存在本机浏览器。

AI 整理名称

用 DeepSeek 把杂乱的物品名批量整理为简洁规范的名称，并自动配上品类图标（会覆盖现有名称）。需先填写并保存上方 API Key。
⚠️ 整理前建议先「导出备份」，以便不满意时恢复。

# 有数·资产记录

📱 一个纯前端 PWA App，记录个人物品，按使用天数摊销成本。

> **核心理念**：每件物品都在替你赚回成本，买得越久，日均成本越低。

## ✨ 特性

- ✅ **单文件架构** - 完整的 `index.html`，无需构建工具
- ✅ **PWA 支持** - 可添加到 iOS/Android 主屏幕使用
- ✅ **完全离线** - 基于 localStorage，无需网络和后端
- ✅ **实时计算** - 每秒自动更新已用天数和日均成本
- ✅ **移动端优先** - 适配 iPhone 安全区域（刘海屏、底部主页键）
- ✅ **深色优雅** - 账簿质感的视觉设计
- ✅ **无依赖** - 不依赖任何框架或 CDN

## 🚀 快速开始

### 本地运行

1. **克隆仓库**
   ```bash
   git clone https://github.com/qiongtaoli-star/youshu.git
   cd youshu
   ```

2. **打开浏览器**
   - 在 VS Code 中使用 Live Server 扩展
   - 或直接用本地 HTTP 服务器（PWA 需要 HTTPS 或 localhost）
   ```bash
   python3 -m http.server 8000
   # 访问 http://localhost:8000
   ```

### 在 iOS 上安装

1. **打开 Safari**，访问部署的网址（GitHub Pages 或自有域名）
2. **点击底部分享按钮** → **添加到主屏幕**
3. **命名为「有数」**（或其他名称）
4. ✅ 完成！现在可以像原生 App 一样使用

### 在 Android 上安装

1. **打开 Chrome**，访问部署的网址
2. **点击菜单** → **安装应用**
3. ✅ 会自动添加到主屏幕

## 📊 核心功能

### 1️⃣ 资产看板

顶部实时显示四项汇总统计：

| 指标 | 说明 |
|------|------|
| **今日总日均成本** | 所有资产的日均成本之和（大数字突出） |
| **资产总数** | 当前记录的物品数量 |
| **累计已用天数** | 所有资产已使用天数之和 |
| **总资产价值** | 所有物品的原始购买价格之和 |

### 2️⃣ 资产明细

按购买日期排序，每行显示：

- **已用天数**（大字号）- 从购买日期到今天
- **物品名称** - 自定义输入
- **购买日期** - YYYY-MM-DD 格式
- **日均成本**（强调色红色）- 原价 ÷ 已用天数
- **原价**（小字）- 购买时的价格

**点击任意一行** → 编辑或删除该资产

### 3️⃣ 添加/编辑

点击右下方 **+ 按钮**，弹出表单：

| 字段 | 类型 | 备注 |
|------|------|------|
| 物品名称 | 文本 | 必填 |
| 购买价格 | 数字 | 元为单位，支持小数 |
| 购买日期 | 日期 | 默认填今天 |

**编辑时** 额外显示 **删除按钮**

## 📖 数据模型

每件资产包含以下字段，存储在浏览器 `localStorage` 中：

```javascript
{
  id: "1719129600000",        // 唯一标识（时间戳）
  name: "iPhone 15 Pro",       // 物品名称
  price: 7999,                 // 购买价格（元）
  date: "2024-06-23"           // 购买日期（YYYY-MM-DD）
}
```

## 🧮 核心计算逻辑

### 已用天数

```
已用天数 = 当前日期 - 购买日期（向下取整，最小为 1）
```

- 每次打开页面时实时重新计算
- 最小值为 1（避免除以 0）
- 每秒自动更新

### 日均成本

```
日均成本 = 购买价格 ÷ 已用天数
```

示例：
- 买 8000 元的手机，第 1 天：日均成本 ¥8000.00/天
- 第 10 天：日均成本 ¥800.00/天
- 第 100 天：日均成本 ¥80.00/天

**理念**：随着使用时间增加，日均成本逐日下降，表示物品在不断替你赚回成本。

## 🔄 数据持久化

- **存储位置**：浏览器 `localStorage`，键名 `youshu_assets`
- **格式**：JSON 数组
- **容量**：通常 5-10 MB（足够存储数千条记录）
- **同步**：不跨设备同步，纯本地存储

### 导出/备份（推荐）

应用内置一键备份，无需开发者工具：

- **导出**：点击列表右上角「导出」，下载 `youshu-backup-YYYY-MM-DD.json` 文件
- **导入**：点击「导入」选择此前导出的 JSON 文件，数据会按 `id` 与现有记录**合并**（相同记录被覆盖），不会清空已有数据

> 建议定期导出备份，避免清理浏览器缓存导致数据丢失。

### 排序

列表右上角的下拉框支持多种排序：最新/最早购买、日均成本高低、使用天数、原价高低，方便快速找出最划算或最“烧钱”的物品。排序偏好会被记住。

## 🐛 故障排除

### PWA 无法安装

- ✅ 确保使用 HTTPS（localhost 除外）
- ✅ 检查 `manifest.json` 是否正确加载（F12 → Network）
- ✅ Service Worker 是否成功注册（F12 → Application）

### 数据丢失

- localStorage 被清理（如清除浏览器缓存）
- 浏览器隐私模式不支持持久化
- **定期在 Console 中导出备份！**

### 页面显示异常

- 硬刷新：`Ctrl+Shift+R`（Windows）或 `Cmd+Shift+R`（Mac）
- 清空 Service Worker 缓存
